1. Sistema Bancário Simples

App em Python com funcionalidades de depósito, saque e extrato.

Este programa é um sistema bancário simples, desenvolvido em Python, com três funcionalidades principais: Depósito: O usuário pode informar o valor do depósito, que será adicionado ao saldo e registrado no extrato. Valores inválidos (menores ou iguais a zero) não são permitidos. Saque: O sistema permite realizar saques com restrições: O valor não pode exceder o saldo disponível. Existe um limite máximo por saque (R$ 500,00 no exemplo). Há um número máximo de saques diários (3 no exemplo). O extrato registra cada saque realizado. Exibir Extrato: Mostra todas as movimentações realizadas (depósitos e saques) e o saldo atual. Caso não haja movimentações, exibe uma mensagem indicando que não foram feitas transações. O programa apresenta um menu interativo no terminal para o usuário escolher a operação desejada: depositar, sacar, consultar extrato ou sair. Ele é estruturado com funções para facilitar a manutenção e reutilização do código. Ao final, o sistema agradece ao usuário por utilizar o serviço.

2. API com SQLITE

Api que utiliza o slqlite

Detalhes: Este projeto implementa uma API REST utilizando FastAPI com um banco de dados SQLite para gerenciar usuários. A aplicação oferece as seguintes funcionalidades: Criação de Usuários: Permite cadastrar novos usuários com os campos nome, e-mail e senha. Verifica se o e-mail já está cadastrado para evitar duplicidades. Listagem de Usuários: Retorna uma lista de todos os usuários cadastrados no banco de dados. Busca por ID: Permite buscar informações de um usuário específico pelo seu ID. Exclusão de Usuários: Remove um usuário do banco de dados com base no ID fornecido. O projeto utiliza SQLAlchemy para modelar o banco de dados e gerenciar operações, além de Pydantic para validação e formatação de dados. A dependência de sessão do banco é tratada com uma função geradora, garantindo o fechamento seguro das conexões. É uma solução simples, eficiente e extensível para gerenciamento de usuários.
